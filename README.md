# Invenza – Billing & Inventory Management System Using MERN Stack

Invenza is a modern web-based Inventory Management System designed to help businesses efficiently manage their products, customers, and analytics from a centralized dashboard.

The application provides a clean interface for tracking inventory data, managing customer information, and viewing business insights through analytics.

---

## Features

• User Authentication (Login / Protected Routes)  
• Dashboard with key inventory insights  
• Product Management (Add, Edit, Delete Products)  
• Customer Management  
• Analytics and reporting page  
• Responsive and modern UI  
• Modular component-based architecture  

---

## Installation and Setup

### 1. Clone the Repository

```
git clone https://github.com/your-username/invenza.git
```

### 2. Navigate to the Project

```
cd invenza/client
```

### 3. Install Dependencies

```
npm install
```

### 4. Run the Development Server

```
npm run dev
```

The application will run at:

```
http://localhost:5173
```

## Application Modules

### Dashboard

Displays an overview of inventory statistics and analytics.

### Products

Allows users to manage product inventory including adding, updating, and removing products.

### Customers

Manage customer details and interactions.

### Analytics

Provides insights and data visualization for inventory operations.
